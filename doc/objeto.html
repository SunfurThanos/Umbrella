Mejora de la etiqueta "align"

La tecnología web fue hecha a los trancazos, por ejemplo a la etiqueta align de HTML le aplicaron una lógica con un rango muy cerrado, es decir si yo a una celda mediante un estilo o clase css modifico el tamaño WxH & luego le inserto la etiqueta align con un valor center, esta ultima etiqueta se rompe, gracias a Umbrella esto llega a su fin, ya que dicha etiqueta no se romperá, ¿Te ha sucedido lo mismo, hay más problemas de este estilo que desees que Umbrella los repare para ti, házmelo saber & estaré encantado de servirte?, esto demuestra el compromiso moral de Umbrella: proteger a los navegadores & usuarios de las tormentas de problemas erráticos.

Mejora de la etiqueta "href"

Esta es otra etiqueta con lógica errática, para hacer corto el cuento, ahora con Umbrella puedes crear interacción de link en celdas tipo texto, botones e imágenes, ademas ahora el nombre de la etiqueta es intuitiva, si deseas que el link a insertar se abra en la misma pestaña debes usar link-main, si deseas que se abra en otra pestaña usa link-tab, ¿estas contento/a con esta mejora, tienes algo que agregar?.

Celdas con etiquetas sincrónicas

Con la nueva etiqueta PasteAttributes, puedes pegar etiquetas de una celda en otras, muy útil para ciertos casos, como por ejemplo menús responsives, donde literalmente a la hora de actualizar los link hay que hacer doble trabajo, con Umbrella eso llega a su FIN.
