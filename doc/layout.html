Super-Flexbox

Este es una de las herramientas más interesante de este apartado, ya que permite crear sistemas de columnas de manera muy sencilla, intuitiva & productiva, solo debes usar la etiqueta hbox, si ha dicha etiqueta no le colocas ningún valor quiere decir que quieres crear un objeto que no sea flexible horizontalmente, ahora bien si le colocas de valor true, el objeto sera flexible lo cual lo hace interesante, en desarrollo web profesional no solo es escribir un código & que sirva, si no que sea lo más simple y auto descriptible, para que los cambios en la pagina web sean fáciles & productivos, por ello esta opción GANA por goleada, ¿que opinas te gusta, tienes una idea que compartir, que haga más fácil la experiencia de desarrollo?.

¿como Umbrella sabe, donde termina o empieza una fila de columnas?: veras las celdas que tengan la etiqueta hbox, serán tomadas como filas, para saber el final de una fila solo basta con "decir": si la celda siguiente es neutral & no tiene la etiqueta mágica, entonces colorin colorado este fila esta cerrada, al detectar la siguiente concurrencia de la etiqueta mencionada, se crea una nueva fila en busca de reclutas para que le hagan compañía.

ADVERTENCIA:&space;&space;las clases CSS a usarse en objetos con (Super FlexBox) deben estar dentro del archivo HTML como código & no como un archivo externo, de lo contrario Umbrella no podrá tomar el control total de la clase..

¿que pasa si las clases están en un archivo?: pues que Umbrella no mejorara ciertas cosas, como por ejemplo para cambiar el margen tienes que usar la sentencia !important, ejemplo: margin-right: 0.9em !important;, en items que tengan un tamaño automático & mediante una clase quieras cambiar las dimensiones en porcentaje o pixeles, debes usar un máximo & no absoluto, ejemplo: max-width: 50%, lo cual no es cosa de otro mundo pero Umbrella te elimina esa leve carga de trabajo.

¿que pasa internamente si las clases están en código del HTML?: para modificar el margen puedes hacerlo de forma normal sin usar ninguna sentencia, para modificar el tamaño de una celda con tamaño automático, puedes usar width o max-width, como vez te pone otro panorama donde el Framework decide que es mas importante o no, ¿algo que agregar?
