BoxComposite

Colocar una celda encima de otra de forma centra u otra posición de composición especifica, suele ser complejo & laborioso no solo codificarlo si no actualizarlo, no te preocupes con este Framework puedes hacerlo de manera sencilla con la etiqueta BoxComposite & asignarle cualquiera de las siguientes posiciones disposibles: top:left, top:center top:right, center:left, center, center:right, bottom:left, bottom:center, bottom:right.

Super-Flexbox

Este es una de las herramientas más interesante de este apartado, ya que permite crear sistemas de columnas de manera muy sencilla, intuitiva & productiva, solo debes usar la etiqueta hbox, si ha dicha etiqueta no le colocas ningún valor o el valor false quiere decir que quieres crear un objeto que no sea flexible horizontalmente, ahora bien si le colocas de valor true, el objeto sera flexible lo cual lo hace interesante, en desarrollo web profesional no solo es escribir un código & que sirva, si no que sea lo más simple y auto descriptible, para que los cambios en la pagina web sean fáciles & productivos, por ello esta opción GANA por goleada, ya que permite crear celdas flexibles y responsives

¿como Umbrella sabe, donde termina o empieza una fila de columnas?: veras las celdas que tengan la etiqueta hbox, serán tomadas como filas, para saber el final de una fila solo basta con "decir": si la celda siguiente es neutral & no tiene la etiqueta mágica, entonces colorin colorado este fila esta cerrada, al detectar la siguiente concurrencia de la etiqueta mencionada, se crea una nueva fila en busca de reclutas para que le hagan compañía.

NOTA:&space;&space;las clases CSS a usarse en objetos con (Super FlexBox) deben estar dentro del archivo HTML como código & no como un archivo externo, de lo contrario Umbrella no podrá tomar el control total de la clase..

¿que pasa si las clases están en un archivo?: pues que Umbrella no mejorara ciertas cosas, como por ejemplo para cambiar el margen tienes que usar la sentencia !important, ejemplo: margin-right: 0.9em !important;, en items que tengan un tamaño automático & mediante una clase quieras cambiar las dimensiones en porcentaje o pixeles, debes usar un máximo & no absoluto, ejemplo: max-width: 50%, lo cual no es cosa de otro mundo pero Umbrella te elimina esa leve carga de trabajo.

¿que pasa internamente si las clases están en código del HTML?: para modificar el margen puedes hacerlo de forma normal sin usar ninguna sentencia, para modificar el tamaño de una celda con tamaño automático, puedes usar width o max-width, como vez te pone otro panorama donde el Framework decide que es mas importante o no.

Hbox-space

Es una herramienta hecha especialmente para Super-Flexbox, la cual permite establecer de manera productiva & asociativa el margen derecho & izquierdo a un determinado item.

También hay una manera más productiva de implementarlo con la etiqueta hbox-layer-space, solo debes crear una celda que encierre a todos los items deseados de una fila.


Diseño web responsive

Umbrella se diferencia notablemente en el diseño responsive, ya que esta inspirado en las simulaciones físicas de los video juegos, actualmente en el desarrollo web tienes que crear reglas Css para calcular un tamaño mínimo o máximo, en resumen un !lio tremendo! para volverlo flexible & actualizarlo.

Para solucionar esto podemos usar las siguientes etiquetas: hbox-isVertical, hbox-isHorizontal, hbox-isWidth, como valor de cada etiqueta podemos crear lógica personalizada gracias a un meta-lenguaje interactivo, este sistema funciona en celdas con Super-Flexbox es decir que tengan la etiqueta hbox, pondré un pequeño ejemplo para ir explicando con mayor detalle esta herramienta.

¿que eso de "row", "grande", ...?: para ocultar o mostrar todos los ítems de una fila, debes usar row, para llamar a una celda cuyo nombre sea grande, pues precisamente debes llamarlo por su nombre.


Funciones responsive
  • hbox-isVertical: detecta cuando una fila de columnas esta en vertical (colition:break*)

  • hbox-isHorizontal: detecta cuando una fila de columnas esta en horizontal

  • hbox-isWidth: detecta el ancho minimo & maximo de una columna

Métodos soportados para los objetos
  • hide: oculta el objeto

  • show: muestra el objeto

  • vertical: coloca todos los ítems de una fila en modo vertical

  • horizontal: pone todos los ítems de una fila en modo horizontal

  • css-class(name_class): podrás cambiar la clase css del objeto, donde dice (name_class) es el nombre de la clase a establecer, si no le asignas ninguna y lo dejas en blanco, le asignara la clase default que tenia la celda.

Esta técnica responsive futurista te permite aprovechar espacio en la pagina de manera flexible & crear cuantas capas de visualización desee, a diferencia de la técnica de la actualidad que malgasta tiempo & espacio en la pagina, ¿te gusto, quieres otro ejemplo?
