Como implemento este FrameWork?

Una vez que descarges Umbrella, dentro del directorio del FrameWork copia la carpeta umbrella al directorio de tu proyecto, en cualquier archivo HTML donde desees utilizar dicha tecnología, pega la siguiente linea de código en la cabecera de dicho archivo, siempre debe ser el primer archivo JS o CSS que importes, para que así Umbrella pueda administrar refactorizado de manera excelente, dicho proyecto tiene que ejecutarse desde un protocolo http como puede ser un localhost para que todas sus herramientas funcionen correctamente.

Hay forma de que funcione sin usar un http?

Por fortuna en cualquier navegador basado en Chrome puedes hacer un truco que te LIBERA como Alicia en el país de las maravillas xD, primero crea un nuevo lanzador de escritorio y ponle el nombre que quieras, en la ruta del programa a usar debes poner la siguiente linea de código...

"$Chrome" --profile-directory=Default --allow-file-access-from-files

Donde dice $Chrome es la ruta del navegador basado en Chrome, una vez creado este lanzador dicho navegador aceptara todas las funciones de Umbrella sin problemas, en resumen tu proyecto funcionara como relojito suizo xD.

En que navegadores funciona?

  • Microsoft Explorer 9/+
  • Microsoft Edge
  • Safari 4/+
  • Google Chrome 4/+
  • Mozilla Firefox 25/+
  • Opera 4/+
  • IPhone/IPad (iOS)

Optimizando el inicio de carga!

Normalmente en HTML creamos una celda y allí colocamos todo el contenido de la pagina web, veras Umbrella, tiene que esperar que se descargue el archivo HTML & se cree el árbol para luego refactorizarlo, eso suele tardar mucho en conexiones lentas o inestables, por ende puedes aplicar una sencilla configuración que harán la espera !agradable!...

A la celda padre, es decir la que contiene todo el contenido web legible, le colocas un nombre name="main_container" y la ocultas style="display: none;", esto permitirá de forma automática que se muestre un icono giratorio alusivo a una barra de carga mientras el DOM esta listo, es decir mientras se van creando los objetos de la pagina & que Umbrella termine de refactorizar los elementos, una vez que eso termine, la celda mencionada se mostrara con un efecto de FADE-IN, es decir una leve animación de opacidad al mostrarse.
