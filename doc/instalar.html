Como implemento este FrameWork?

Una vez que descarges Umbrella, dentro del directorio del FrameWork copia la carpeta umbrella al directorio de tu proyecto, en cualquier archivo HTML donde desees utilizar dicha tecnología, pega las siguientes lineas de código en la cabecera del archivo mencionado.

En que navegadores funciona?

Testeado al 100% en los siguientes navegadores:
  • Microsoft Edge
  • Google Chrome
  • Mozilla Firefox
  • Opera

NOTA: este FrameWork no es compatible con versiones de Firefox anteriores a la 35 u otros navegadores con versiones obsoletas, Internet explorer tampoco es compatible por muchas razones, entre ellas que ya Microsoft no le interesa seguir desarrollándolo, la muerte oficial de este navegador sera el 17 de agosto del año 2021, ¿que pasa si un usuario abre mi web hecha con Umbrella en un navegador no soportado?: aparecerá un mensaje amigable solicitándole al usuario que descargue e instale el navegador Firefox o Chrome, por ejemplo si quieres usar a Firefox, la versión 35 es el requerimiento mínimo para navegar por la web de manera estándar, usar versiones anteriores dañan notablemente la experiencia aun si no usas este FrameWork, en computadoras viejas con Windows XP esta versión de Firefox funciona correctamente, así que no tienes ningún problema por la cual no usar Umbrella & disfrutar tu & tus usuarios de sus ventajas.
